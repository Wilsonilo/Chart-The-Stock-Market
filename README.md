### About

Simple Web-app for the challenge "Chart the Stock Market" from freecodecamp.org:

https://www.freecodecamp.org/challenges/chart-the-stock-market

Works with COINS from the Blockchain on the API: https://github.com/CoinCapDev/CoinCap.io 

Uses Chartjs, jQuery and animatedCss and Bootstrap.
